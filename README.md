# TestCode
A python library that allows you to easily test functions in your code

## Authors
- [Fort-P](https://github.com/fort-p)

## Installation
Install with pip:
```powershell
pip install TestCode
```

## Usage
Import the code at the top of your file:
```python
from TestCode.TestCode import testCode
```
